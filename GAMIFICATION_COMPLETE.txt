RANK SYSTEM DETAILS
===================

Rank Đồng       (Bronze)       - Levels 0-10
Rank Sắt        (Iron)         - Levels 11-20
Rank Vàng       (Gold)         - Levels 21-30
Rank Lưu Ly     (Lapis Lazuli) - Levels 31-40
Rank Lục Bảo    (Emerald)      - Levels 41-50
Rank Kim Cương  (Diamond)      - Levels 51-60
Rank Thạch Anh  (Amethyst)     - Levels 61-70 (MAX)


XP REWARD SYSTEM
================

Base: 100 XP per workout
Per Exercise: 20 XP each
Difficulty Bonus:
  - Low: 0 XP
  - Medium: 30 XP
  - Hard: 60 XP
Consistency Bonus: 50 XP (when streak maintained)
Nutrition Bonus: 30 XP (when nutrition logged)

Example: 5 exercises + hard + nutrition + streak = 360 XP
